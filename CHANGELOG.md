# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-12

### Добавлено
- Базовый класс `LifeCycle` для управления жизненным циклом хуков.
- Абстрактный класс `BaseExecutableHook` с методами `_do_init` и `_do_quit`.
- Класс `HookAdapter` для адаптации произвольных вызываемых объектов в хуки.
- Поддержка зависимостей между хуками (раздельно для инициализации и завершения).
- Топологическая сортировка зависимостей с обнаружением циклов.
- Состояния хуков: `NEW`, `INITIALIZED`, `FINALIZED`, `ERROR`.
- Обработка критических ошибок (`FATAL`) с остановкой всего процесса и некритических (`FAILURE`) с продолжением.
- Возможность передать пользовательский `shutdown_callback` вместо жёсткого вызова `sys.exit`.
- Методы `add_hook`, `remove_hook`, `run`, `reset` в `LifeCycle`.
- Полноценная документация кода: все публичные методы и классы снабжены docstring в формате Google Style с разделами `Args`, `Returns`, `Raises`, `Examples`.
- Аннотации типов (PEP 484) для всех функций и методов.
- Логирование на русском языке с использованием модуля `logging` (формат: время, имя логгера, уровень, сообщение, кодировка UTF-8).
- Иерархия пользовательских исключений:
  - `LifecycleError` – базовый класс.
  - `LifecycleConfigError` – ошибки конфигурации.
  - `LifecycleStateError` – ошибки перехода между состояниями.
  - `LifecycleDataError` – ошибки в данных.
  - `LifecycleResourceError` – ошибки доступа к ресурсам.
- Все исключения имеют сообщения на русском языке с контекстом.
- Примеры использования в docstring (doctest) и отдельные файлы в директории `examples/`.
- Файл `README.md` с описанием установки, быстрого старта, API и примеров.
- `CONTRIBUTING.md` с правилами разработки, требованиями к коду (русские логи, иерархия исключений, стиль документации).
- `Makefile` для автоматизации тестов, линтинга, форматирования и проверки типов.
- Настройки `ruff` и `mypy` для статического анализа.
- Лицензия MIT.
